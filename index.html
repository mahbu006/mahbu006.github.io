Fruit Detector Project

EE2361 Group Project. Developed an API for an interactive device implemented through the PIC24FJ64GA002 Microcontroller. When a specific fruit is touched, it is relayed through the CAP1188 sensors, and a customized animation will be displayed on the RGB LED 8x8 matrix through enabling an appropriate bit stream.
